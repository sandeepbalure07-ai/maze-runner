# Maze Runner

Live Demo:
https://sandeepbalure07-ai.github.io/maze-runner/

A browser-based maze game built using HTML, CSS, and JavaScript. The game generates random mazes and tracks player score, moves, completion time, and best score.

## Features

- Random maze generation
- Real-time score tracking
- Move counter
- Timer system
- Best score tracking
- Keyboard controls
- Maze completion detection
- Responsive game interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Canvas API

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Start playing.

## Controls

- Arrow Keys → Move player
- Reach the goal cell to complete the maze
